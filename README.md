DOCUMENTATION TECHNIQUE : API DE MODÉRATION AUTOMATISÉE1. Présentation du ProjetCette API fournit un système complet de gestion de contenu avec analyse de toxicité en temps réel. Elle intègre le modèle de Deep Learning Detoxify pour automatiser la détection de contenus inappropriés et permettre une intervention humaine ciblée.2. Architecture TechniqueFramework : Django 5.x / Django REST FrameworkMoteur d'IA : Detoxify (basé sur l'architecture Transformer)Authentification : Token-based AuthenticationBase de données : SQLite (environnement de développement)3. Installation et ConfigurationPrérequisPython 3.10+Pip (gestionnaire de paquets)Procédure d'installationEnvironnement virtuel :python -m venv venvsource venv/bin/activate (ou venv\Scripts\activate sur Windows)Installation des dépendances :pip install django djangorestframework django-cors-headers detoxify torchInitialisation de la base de données :python manage.py makemigrationspython manage.py migrateCréation du compte administrateur :python manage.py createsuperuser4. Spécifications du Système de ModérationL'analyse est déclenchée automatiquement via un signal Django lors de la sauvegarde d'un objet Message.ParamètreSeuil / ValeurActionSeuil de Toxicité> 0.7Statut : "flagged" (quarantaine)Seuil de Signalement>= 3 reportsStatut : "flagged" (automatique)Catégories analyséesToxicity, Insult, Threat, ObsceneEnregistrement dans ModerationResult5. Guide des Endpoints APIA. Gestion des MessagesPOST /api/messages/Usage : Soumission d'un nouveau contenu.Payload : {"content": "string"}POST /api/messages/{id}/report/Usage : Signalement d'un message par un utilisateur.Payload : {"reason": "string", "description": "string"}B. Interface de Modération (Admin uniquement)GET /api/moderation/pending/Usage : Liste des messages en attente de révision (statuts pending ou flagged).POST /api/moderation/{id}/approve/Usage : Validation manuelle d'un message.Payload : {"reason": "string"}POST /api/moderation/{id}/reject/Usage : Rejet définitif et masquage du contenu.Payload : {"reason": "string"}C. AnalytiqueGET /api/moderation/stats/Usage : Récupération des indicateurs de performance (KPI) de la plateforme.
